# マーケティング検定1級 演習道場

日本マーケティング協会「マーケティング検定1級」（記述式・CBT）の個人学習アプリ。
単一HTMLで動作し、サーバー・外部ライブラリ不要。PC・スマートフォン両対応。

## 使い方

- `marken1-dojo.html` をブラウザで開くだけで動作します（`index.html` は同ファイルへの転送）
- GitHub Pages で公開すると、PC・スマホから同じURLでアクセスでき、
  学習進捗はブラウザ内（localStorage / IndexedDB）に**恒久保存**されます
- iPhoneでは Safari で開き「共有 → ホーム画面に追加」するとアプリのように使えます

## 機能

- 記述演習37問（採点ポイント自己採点・キーワード自動チェック・S/A得点区分換算・Claude採点プロンプト）
- 昨年度問題の復元＋公式模擬テストの傾向分析
- 用語ドリル66語（逆想起モード）／穴埋めドリル28問
- 音声学習（端末内蔵の音声合成・話し言葉の専用原稿・オフライン動作）
- 論点マップ（73論点の自信度セルフチェック）／直前総まとめシート
- 模試モード（4題120分タイマー・400点換算）／連続学習日・今日のミッション／周回管理

## 進捗データについて

- 学習進捗は**各端末のブラウザ内**に保存されます（このリポジトリには含まれません）
- 端末間の引き継ぎは、アプリ内「進捗をファイルに保存」→ もう一方の端末で「復元」
- ブラウザの「閲覧履歴・サイトデータの削除」を行うと消えるため、定期的なバックアップを推奨
